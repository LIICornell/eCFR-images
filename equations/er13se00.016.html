R V = ( d y ) × ( C ) + F 1 + ( r y ) × ( i ) − A I

(Equation 37)

WHERE:
RV = Redemption value
F = Face amount redeemed
AI = Accrued interest = [ ( d − r ) / y ] × C
d = Number of days from original issue of the certificate of indebtedness to its maturity date
r = Number of days from redemption date to the certificate of indebtedness' maturity date
y = 365. if the number of days in the year following issue of the certificate of indebtedness does not include a leap year day; 366, if the number of days following issue of the certificate of indebtedness does include a leap year day
i = Treasury borrowing rate over the remaining term to maturity, expressed in decimals
C = The regular annual interest
