(1) The tests required by this table apply to a fully assembled percussion-activated device including all internal ordnance. (2) This test must subject each percussion-activated device sample to the qualification environmental test level. (3) A high-temperature storage test is optional. A lot will have an initial service-life of three years if it passes this test and all the required tests. A lot will have an initial service-life of one year if it passes all the required tests, but does not undergo this test. (4) The lot sample quantity must be no less than the greater of 10% of the lot or nine sample units.
