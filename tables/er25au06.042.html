(1) Each sample safe-and-arm device to undergo qualification testing must first successfully complete all acceptance tests required by table E417.25-1. (2) A component must undergo this test before the first and after the last operating environment test. (3) A component must undergo this test during each operating environment test. (4) One safe-and-arm device must undergo the extended stall and abnormal drop tests designated with an X. (5) The same six sample safe-and-arm devices must undergo each test designated with an X. For a test designated with a quantity of less than six, each safe-and-arm device tested must be one of the original six sample components. (6) One safe-and-arm device must undergo the containment test and two safe-and-arm devices must under go the barrier functionality test. The safe-and-arm device samples used for these tests need not be flight safe-and-arm devices. The test samples must duplicate all dimensions of a flight safe-and-arm device, including gaps between explosive components, free-volume, and diaphragm thickness. (7) An unsealed component that has successfully completed salt-fog, humidity, fungus resistance, and fine sand qualification test need not undergo a leakage test.
